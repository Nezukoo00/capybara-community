💾 ХРАНЕНИЕ ДАННЫХ:

Используется localStorage браузера:
  • users - список всех зарегистрированных пользователей
  • comments - все комментарии со всех страниц
  • currentUser - текущий вошедший пользователь

Структура данных:
  Users:
    - id (уникальный идентификатор)
    - username (имя пользователя)
    - email (электронная почта)
    - password (пароль)
    - joinDate (дата регистрации)

  Comments:
    - id (уникальный идентификатор)
    - userId (ID автора)
    - username (имя автора)
    - text (содержание комментария)
    - date (дата публикации)
    - page (страница: "home" или "about")

🚀 КАК ЗАПУСТИТЬ:

1️⃣ ВАРИАНТ 1 - Просто открыть в браузере:
   • Откройте файл index.html двойным кликом
   • Или откройте любой HTML файл в браузере

2️⃣ ВАРИАНТ 2 - С локальным сервером (рекомендуется):

   Если у вас Python:
   $ cd c:\Users\админ\gemdev
   $ python -m http.server 8000
   Затем откройте: http://localhost:8000

   Если у вас Node.js:
   $ npx http-server -p 8000
   Затем откройте: http://localhost:8000

3️⃣ ВАРИАНТ 3 - Через VS Code Live Server:
   • Установите расширение "Live Server"
   • Кликните правой кнопкой на index.html
   • Выберите "Open with Live Server"

📋 ПРИМЕР ИСПОЛЬЗОВАНИЯ:

1. Откройте главную страницу
2. Нажмите "Зарегистрироваться"
3. Введите:
   - Имя: Капибара
   - Email: capybara@example.com
   - Пароль: password123
4. Нажмите "Зарегистрироваться"
5. Вы автоматически войдете в аккаунт
6. На главной странице появится форма комментариев
7. Напишите комментарий и нажмите "Отправить"
8. Комментарий появится в ленте
9. Нажмите "Профиль" чтобы увидеть свои комментарии
10. Вы можете удалить свой комментарий нажав "Удалить"
